# Responsive Art Landing Page 

This project is part of a step-by-step YouTube tutorial in which we build a responsive art landing page from scratch using HTML, CSS and JavaScript.

![preview img](/preview.png)

## 📌 Project Features
- Fully responsive art home landing page (Mobile First Methodology).
- Clear and semantic HTML structure.
- Use of CSS variables and modern JavaScript functions.
- Smooth animations with GSAP.
- Elegant glides with Swiper Js.
- Compatible with all modern browsers and devices.
